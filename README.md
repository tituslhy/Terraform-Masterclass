## Installation
I use a Mac!

To install Terraform
```
brew tap hashicorp/tap
brew install hashicorp/tap/terraform
```

To install AWS CLI
```
brew install awscli
```

To login to AWS CLI in a terminal session:
```
source .env
```

This loads the env vars in the .env file
